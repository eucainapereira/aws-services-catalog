# ☁️ AWS Services Catalog & Architecture Library

![AWS Services Catalog](icons/icon128.png)

Uma extensão do Chrome poderosa, elegante e profissional projetada para arquitetos, desenvolvedores e estudantes de cloud. Explore o vasto ecossistema AWS com uma interface premium e aprenda através de arquiteturas de referência do mundo real.

## 🚀 Funcionalidades Principais

### 📚 Catálogo Completo de Serviços
- **+200 Serviços Listados:** Uma enciclopédia completa da AWS na ponta dos seus dedos.
- **17 Categorias Oficiais:** Navegação organizada por Compute, Storage, Database, AI/ML, Security, IoT e muito mais.
- **Modais Detalhados:** Cada serviço inclui:
    - Definição técnica clara.
    - Casos de uso práticos.
    - Listagem de sub-serviços e componentes internos.
    - Links diretos para a documentação oficial.

### 🏗️ Biblioteca de Arquiteturas de Referência
Acesse padrões de arquitetura prontos para uso, desde o básico até o nível enterprise:
- **Starter Patterns:** Sites estáticos, blogs em WordPress e MVPs rápidos.
- **Modern Architectures:** Serverless APIs, Microserviços com Kubernetes e Web Apps 3-Tier.
- **Advanced & Enterprise:** IA Generativa com Bedrock, Data Lakes, MLOps e Arquiteturas Zero Trust.
- **Niche Solutions:** IoT Industrial, Backend de Jogos Multiplayer e Pipelines de Vídeo (VOD).

### ✨ Experiência de Usuário Premium
- **Design Moderno:** Interface com efeitos de glassmorphism, suporte nativo a modo escuro e animações fluidas.
- **Busca Inteligente:** Localização instantânea de serviços por nome, sigla ou funcionalidade.
- **Navegação Intuitiva:** Troca rápida entre a visão de Catálogo e a visão de Arquiteturas.
- **Filtros Dinâmicos:** Explore serviços por categoria com feedback visual em tempo real.

## 🛠️ Tecnologias Utilizadas

- **HTML5 Semantic Structure**
- **CSS3 Modern Design System** (Vanilla CSS com variáveis e animações otimizadas)
- **JavaScript (ES6+)**
- **Chrome Extension API v3**

## 📦 Como Instalar (Modo Desenvolvedor)

1. Faça o download ou clone este repositório.
2. Abra o Google Chrome e acesse `chrome://extensions/`.
3. Ative o **Modo do desenvolvedor** (canto superior direito).
4. Clique em **Carregar sem compactação** e selecione a pasta raiz deste projeto.

## 📊 O que há no Catálogo?

A extensão cobre todas as verticais principais da AWS:
- **Computação:** EC2, Lambda, EKS, Fargate, App Runner...
- **Armazenamento:** S3, EBS, EFS, FSx (ONTAP/Lustre/Windows)...
- **Banco de Dados:** RDS, Aurora, DynamoDB, ElastiCache, Redshift...
- **IA & Machine Learning:** Bedrock, SageMaker, Rekognition, Polly...
- **Segurança & Rede:** VPC, CloudFront, IAM, Shield, WAF, Control Tower...
- **E muito mais!**
